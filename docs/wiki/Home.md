# Moonrise-MC Wiki: Home / Overview

Welcome to the official documentation for **Moonrise-MC**, a high-performance Minecraft Paper 26.1.2 server fork tailored for optimized mechanics and unparalleled efficiency.

## Project Overview

Moonrise-MC is built on top of PaperMC for version 26.1.2, implementing extensive performance modifications including an advanced custom redstone engine, asynchronous mechanics, and Java 25 compatibility out-of-the-box. The project is designed for server administrators and developers seeking uncompromising performance, particularly on modern hardware and containerized environments.

## Related Projects

- **Moonrise Dashboard**: A companion web dashboard designed to integrate natively with Moonrise-MC servers. It is built using modern frontend technologies, specifically **React Native Expo** and **Nativewind v5**, offering a responsive, cross-platform server management interface.
